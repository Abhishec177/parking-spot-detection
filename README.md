### Parking Spots Detection  (VLSI Lab Project)

Contents:

This repostiory contains an end-to-end working model of a parking spot detection application which recieves images of parking areas to show which spots are vacant.

- Contains the front-end and server code to run the application
- Contains the code required to build, train, test and apply the model to the web application.
